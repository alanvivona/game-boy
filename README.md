# gb-asm

## A Gameboy assembler

This installs the cli `gb-asm`

### Usage  
```
   -u              interactive mode  
   -h --help       help message  
```

Examples:
> echo "LD A,A" | gb-asm

### Install using NPM
> npm install -g game-boy-assembler

### Run using Docker:  
> docker run -ti alanvivona/gameboy  

Link to DockerHub image: https://hub.docker.com/r/alanvivona/gameboy
